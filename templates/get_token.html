What is an Access Token?

An access token is required to use the URL Checker service. This token ensures that only authorized users can access the service.

Tokens are generated by our Telegram bot and typically valid for 24 hours.

How to Get a Token

Follow these steps to get your access token:

1. Open Telegram and message our bot at the link below

2. Send the command /token to the bot

3. Copy the token you receive and paste it below
